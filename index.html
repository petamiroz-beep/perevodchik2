Информация о переводчике

Это демо-версия переводчика. Для работы с реальным API необходимо:

  1. Получить API ключ от сервиса перевода (например, Yandex Translate, Google Translate)
  2. Заменить функцию translateText() на код, отправляющий запрос к API
  3. Убедиться, что CORS политика позволяет выполнять запросы
